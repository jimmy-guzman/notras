# Next.js Starter

## Getting Started

### Turso

```
brew install tursodatabase/tap/turso
```

```
turso auth login
```

```
turso db create pilas
```

```
turso db create pilas-dev --from-db pilas
```

```
 turso db tokens create pilas-dev
```

```
turso db show pilas-dev --url
```

### Database

```
pnpm db:push
```
